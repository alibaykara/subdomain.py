## Run Locally

Clone the project

```bash
  git clone https://github.com/alibaykara/subdomain.py.git
```

Go to the project directory

```bash
  cd subdomain.py
```

Start the python3

```bash
  python3 subdomain.py -d hackerone.com -o subdomain.txt
```
